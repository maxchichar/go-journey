Write a function that ask a user to input a word then ask a user to select an operation by either entering lastword or capitalize or delete index.

if `lastword` inputted by user, the program should return the last character on the word inputted.

if `capitalize` inputted by the user, the program should return the word capitalized

if `deleteIndex` inputted by the user, the program should as the user to input index, when index inputted, the character at that index should be deleted and the rest of the letters in the word be returned.

else if the index inputted is greater than the length of the word, the program should display an error message and restart that delete index block of code
